# WatchtimeCV

**WTCV** é uma aplicação desenvolvida para empresas que desejam obter um parâmetro real e preciso de quantas pessoas estão olhando para suas propagandas expostas nas cidades.

O sistema utiliza visão computacional para detectar e contabilizar o número de olhares direcionados a cada anúncio, permitindo análises de engajamento, alcance e efetividade das campanhas publicitárias em tempo real.

Com o WatchtimeCV, as empresas podem mensurar o impacto visual de outdoors, painéis digitais e outras formas de mídia urbana, otimizando seus investimentos em marketing e publicidade.
